# WebService
<h1>Instalation</h1>
Ce placer dans le dossier puis faire:
 <code>npm install</code>

<h1>Tests Unitaires</h1>
Commande pour lancer les tests unitaires
<code>npm test</code>

<h1>Lancer Angular</h1>
Ce placer dans le dossier
<code>ng serve --open</code>
